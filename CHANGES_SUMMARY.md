| Колонка | Описание |

| № | Номер строки |
| User ID | ID пользователя в Telegram |
| Ник | Username (если есть) |
| Имя | First name (если есть) |
| Дата присоединения | Дата первого старта бота |
| Баланс | Текущий баланс |

## 🚀 Использование

### Для администратора:
1. Нажмите "📊 Статистика" в админ-меню
2. Выберите "📦 Экспорт БД" для полного экспорта
3. Найдите пользователя через "🔍 Поиск пользователя"
4. Нажмите "📦 Покупки" для просмотра истории
5. Нажмите на товар для просмотра всех деталей (с session и tdata)
