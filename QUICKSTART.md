## 📖 Основные команды

### 👨‍💼 Вы (администратор):

```
/add         Добавить аккаунт
/addcode     Добавить код к существующему аккаунту
/addc        Добавить страну (Узбекистан, Египет и т.д.)
/adddel      Удалить страну
/acclist     Все аккаунты по странам
/list        Балансы пользователей
/mn          Выдать денег пользователю
```

### 🛍️ Пользователи:

Используют инлайн кнопки в чате:
- **🛍️ Купить** - выбрать страну → аккаунт → купить
- **💰 Баланс** - показать текущий баланс
- **📜 История** - последние транзакции
- **📧 Обратная связь** - написать админу

```
1. /addc

2. /add
   → Выбирите: Узбекистан
   → Телефон: +998901234567
   → Код: 12345
   → 2FA: recovery_code
   → Цена: 500
   ✅ Аккаунт добавлен

3. Пользователь пишет: /start
   → Нажимает: 🛍️ Купить
   → Выбирает страну и аккаунт
   → Система проверяет баланс
   → Если денег достаточно → показывает коды
   → Аккаунт помечен как проданный
   ✅ Деньги перечислены

4. /list
   → Видите текущие балансы всех пользователей
```

## 📊 Файлы данных

Создаются автоматически в папке `data/`:

- `accounts.json` - все аккаунты
- `balances.json` - деньги пользователей
- `transactions.json` - история
- `countries.json` - страны
- `feedback.json` - обратная связь
- `users.json` - пользователи
